# Feature: Radio Browser & Library

## Summary
A service and UI for discovering and saving online radio stations. Users can browse by tags/categories, search for specific stations, and add them to their local library.

## Rationale
Provides a modern alternative to local files by integrating internet radio discovery without requiring external browser interaction.

## Expected Behavior
- **Browser**: Search and list stations via the Radio-Browser API.
- **Discovery**: Advanced filters for Country and Language with persistent background sync.
- **Popularity**: Display community vote counts on all stations.
- **Now Playing**: Prominent display of live ICY metadata and links to station homepages.
- **Library**: Save stations to the local database.
