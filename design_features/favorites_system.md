# Feature: Favorites & Liked Music

## Summary
A global system for marking content as 'Favorite'.

## Rationale
Standard user expectation for quick access to preferred content.

## Expected Behavior
- **Tracks**: Toggle 'Favorite' (rating: 1) to automatically appear in the 'Likes' smart playlist.
- **Albums/Artists**: Toggle favorite status for library filtering.
- **Smart Playlist**: A dynamic view that query-aggregates all favorited content.
